Planeter

Orbit based puzzle Game

The Project

The project aims to create an orbit based puzzle game on mobile devices with high influence from the desktop game Kerbal Space Programm. For the static orbit prediction of planets Keplerian elements and their laws of planetary motion are used. The dynamic predictions for the user-controlled spaceship use a performance adapted gravity system approach and simple Newtonian calculations.

Design Pillars

To reduce the risk of feature creeping three design pillars have been defined that need to be addressed and may not be contradicted by additional features or content.

  • minimalism
  • education
  • gratification

Gameplay

The game is seperated into three stages missions, staging and the game.

  • In the mission phase the user can select a mission, such as the Sputnik mission, which sets the goal for the player. Reaching the the goal rewards the player.
  • In the staging phase the player sets up the rocket. The staging cannot be changed during the game, thus the stages need to be selected carefully.
  • Finally the player is able to play out the set mission with the already set staging
